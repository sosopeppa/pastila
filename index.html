Блог: Расшифровка шифров Цезаря

Решены следующие задачи на определение ключа шифра Цезаря по известным парам «открытый текст — шифротекст».

Задача а: АПЕЛЬСИН → САЦЬНВЩЮ

При анализе этой пары выяснилось, что невозможно подобрать единый ключ сдвига для всего слова в стандартном русском алфавите (32 буквы без «Ё»). Разные буквы дают разные значения ключа (16, 17, 18), что противоречит принципу шифра Цезаря.

Вывод: либо в шифротексте допущена ошибка, либо использован нестандартный алфавит. Единого ключа не существует.

Задача б: АБРИКОС → ЫЬЛГЕЙМ

При использовании 32-буквенного алфавита (А–Я без «Ё»):

А(0)  +27 → Ы(27) ✅  
Б(1)  +27 → Ь(28) ✅  
Р(17) +27 → Л(12) ✅ (44 mod 32 = 12)  
И(8)  +27 → Г(3)  ✅ (35 mod 32 = 3)  
К(10) +27 → Е(5)  ✅ (37 mod 32 = 5)  
О(14) +27 → Й(9)  ✅ (41 mod 32 = 9)  
С(18) +27 → Н(13) ❌ (в шифротексте указано «М» = 12)

Шесть из семи букв совпадают при ключе k = 27. Последняя буква, вероятно, содержит опечатку (должно быть «Н», а не «М»).

Вывод: наиболее вероятный ключ — k = 27.

Итог

  • Пара (а): нет единого ключа — возможна ошибка в данных.
  • Пара (б): ключ шифра Цезаря равен 27 (с высокой вероятностью).

Расшифровка сообщений, зашифрованных шифром Цезаря

Даны два шифротекста, зашифрованные с неизвестным ключом \( k \), где \( 0 < k < 32 \). Используется стандартный русский алфавит из 32 букв (без «Ё»):

А Б В Г Д Е Ж З И Й К Л М Н О П Р С Т У Ф Х Ц Ч Ш Щ Ъ Ы Ь Э Ю Я

а. Шифротекст: ФХНЗКЧ

После перебора всех возможных ключей найден осмысленный результат:

При k = 5:

  • Ф → П
  • Х → Р
  • Н → И
  • З → В
  • К → Е
  • Ч → Т

Результат: ПРИВЕТ

Ключ: \( k = 5 \)

б. Шифротекст: ЦЩЕБФ

Полный перебор ключей не дал общеупотребительного русского слова. Наиболее близкий вариант при том же ключе \( k = 5 \):

ЦЩЕБФ → СФАЬП (бессмысленно).

Возможные причины:

  • Опечатка в шифротексте,
  • Использовано редкое или вымышленное слово,
  • Ошибка в условии задачи.

Если предположить, что оба сообщения зашифрованы одним ключом, то ключ \( k = 5 \), но открытый текст второго сообщения остаётся неопределённым.

Заключение

Первое сообщение успешно расшифровано как «ПРИВЕТ» с ключом 5. Второе сообщение требует уточнения условия.
